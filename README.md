# Smart Contract Assistant

A **local AI-powered assistant** for document-based question answering (RAG pipeline) using **Ollama**, **LangChain**, **FAISS**, and **Gradio/FastAPI**.  
Supports **PDF & DOCX files**, retrieval-augmented QA, and chat interface with conversation history.
